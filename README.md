Pendu

Jeu du Pendu en Python avec interface graphique, images et sons.

Description

Ce projet est une implémentation du jeu du pendu.
Le joueur doit deviner un mot lettre par lettre avant d’avoir perdu toutes ses vies.
Le jeu utilise des images, de la musique et une structure en classes.

Lancer le jeu

Avoir Python 3 installé

Cloner le dépôt :

git clone https://github.com/MilanM-V/Pendu.git
cd Pendu


Lancer le jeu :

python main.py

Structure du projet
Pendu/
├── classe/     # Classes du jeu
├── donnee/     # Données (mots, etc.)
├── image/      # Images du jeu
├── musique/    # Sons et musiques
├── main.py     # Fichier principal
└── README.md

Technologies utilisées

Python
